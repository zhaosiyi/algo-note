归并排序&逆序对

创建日期：2016-10-24

简介

归并排序可以顺便求出逆序对个数。

代码

// array 为原数组，[beg,end) 为排序的范围，temp 为辅助数组，大小不小于 array 。
// 如果需要求逆序对的话，初始化 cnt=0 。
void merge_sort(int *array, int beg, int end, int *temp){
    if(end-beg<=1) return; // 序列长度小于等于1，无需排序。
    int mid= (beg+end)/2; // 取得中间位置。beg+end 可能溢出，改为 mid= beg+(end-beg)/2 可避免溢出。但通常无需这么做。
    merge_sort(array, beg, mid, temp); // 排序左半区间。
    merge_sort(array, mid, end, temp); // 排序右半区间。
    // 开始合并。b1 为左半区间的开头，b2 为右半区间的开头，i为temp中保存结果的位置。
    int b1= beg, b2= mid, i=beg;
    while(b1<mid||b2<end){ // 当两个序列至少一个不为空。
        // 如果右半区间为空 or (左半区间不空 and 左半区间头部元素更小)
        if(b2>=end||(b1<mid&&array[b1]<=array[b2])) temp[i++]= a[b1++]; // 复制左半区间首元素到结果。
        else{
            temp[i++]= array[b2++]; // 将右半区间首元素到结果。
            cnt+= mid-b1; // 求逆序对只需加这一行代码。
        }
        for(int i=beg; i<end; ++i){
            array[i]= temp[i]; // 将结果拷贝回原数组。
        }
    }
}
